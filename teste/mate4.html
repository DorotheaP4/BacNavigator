SUBIECTUL I (30 de puncte)

5p

1. Determinați numărul real x pentru care numerele 2, x+1, 8 sunt termeni consecutivi ai unei progresii geometrice.

5p

2. Se consideră funcția f : R → R, f(x) = 3x - 5. Calculați f(1) + f(2) + f(3).

5p

3. Rezolvați în mulțimea numerelor reale ecuația log₃(x² - 8) = 1.

5p

4. Calculați C₅² + A₄².

5p

5. În reperul cartezian xOy se consideră punctele A(1, 1), B(3, 5) și C(5, 1). Arătați că triunghiul ABC este isoscel.

5p

6. Se consideră vectorii →u = 2→i - 3→j și →v = →i + a→j, unde a este număr real. Determinați numărul real a pentru care vectorii →u și →v sunt perpendiculari.

SUBIECTUL al II-lea (30 de puncte)

5p

1. Se consideră matricea A(x) = [[1+x, x], [x, 1+x]], unde x este număr real.

5p

a) Arătați că det(A(1)) = 3.

5p

b) Demonstrați că A(x) - A(y) = (x-y)A(0), pentru orice numere reale x și y.

5p

c) Determinați numerele reale x pentru care A(x) · A(-x) = I₂, unde I₂ = [[1, 0], [0, 1]].

5p

2. Se consideră polinomul P(X) = X³ - 2X² - 2X + 1.

5p

a) Arătați că P(1) = -2.

5p

b) Determinați câtul și restul împărțirii polinomului P la X - 2.

5p

c) Știind că x₁, x₂, x₃ sunt rădăcinile polinomului P, calculați x₁x₂ + x₁x₃ + x₂x₃.

SUBIECTUL al III-lea (30 de puncte)

5p

1. Se consideră funcția f : R → R, f(x) = (x² + 1)ex.

5p

a) Arătați că f'(x) = (x² + 2x + 1)ex, x ∈ R.

5p

b) Determinați ecuația asimptotei spre -∞ la graficul funcției f.

5p

c) Demonstrați că f(x) ≥ 1 pentru orice x ∈ R.

5p

2. Se consideră funcția f : R → R, f(x) = cos x + x.

5p

a) Arătați că ∫0π/2 (f(x) - x) dx = 1.

5p

b) Calculați ∫01 x · cos x dx.

5p

c) Calculați ∫-11 (f(x) - x)² dx.

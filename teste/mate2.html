SUBIECTUL I (30 de puncte)

5p

1. Determinați rația progresiei geometrice (bn)n≥1, știind că b₂ = 10 și b₄ = 250.

5p

2. Se consideră funcțiile f : R → R, f(x) = x+1 și g : R → R, g(x) = x²-2. Determinați (f ο g)(x).

5p

3. Rezolvați în mulțimea numerelor reale ecuația log₂(x+1) = 3.

5p

4. Calculați numărul de moduri în care se poate forma o echipă de 3 elevi dintr-un grup de 7 elevi.

5p

5. În reperul cartezian xOy se consideră punctele A(1, 2) și M(3, 4). Determinați coordonatele punctului B, știind că M este mijlocul segmentului AB.

5p

6. Se consideră triunghiul ABC cu AB = 3, AC = 5 și BC = 7. Calculați cosinusul unghiului B.

SUBIECTUL al II-lea (30 de puncte)

5p

1. Se consideră matricea M(x) = [[x, 1], [-1, x]], unde x este număr real.

5p

a) Arătați că det(M(0)) = 1.

5p

b) Arătați că M(x) · M(y) = M(x+y) + (xy-1)I₂, pentru orice numere reale x și y, unde I₂ = [[1, 0], [0, 1]].

5p

c) Determinați numerele reale x pentru care M(x) · M(x) = M(2x).

5p

2. Se consideră polinomul P(X) = X³ - X² + mX - m, unde m este număr real.

5p

a) Arătați că P(1) = 0, pentru orice număr real m.

5p

b) Pentru m=1, determinați rădăcinile polinomului P.

5p

c) Determinați numărul real m pentru care 1/x₁ + 1/x₂ + 1/x₃ = 1, unde x₁, x₂, x₃ sunt rădăcinile polinomului P.

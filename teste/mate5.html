5p

2. Se consideră funcțiile f : R → R, f(x) = x + 2 și g : R → R, g(x) = x². Calculați (g ο f)(-1).

5p

3. Rezolvați în mulțimea numerelor reale ecuația √(x + 2) = x.

5p

4. Determinați câte numere naturale pare de trei cifre distincte se pot forma cu elementele mulțimii {0, 1, 2, 3, 4}.

5p

5. În reperul cartezian xOy se consideră dreapta d de ecuație y = 2x + 3. Calculați distanța de la originea O(0, 0) la dreapta d.

SUBIECTUL al II-lea (30 de puncte)

5p

1. Se consideră matricea A(x) = [[x, x+1], [x-1, x]], unde x este număr real.

5p

a) Arătați că det(A(2)) = 3.

5p

b) Arătați că A(x) + A(y) = A(x+y) + A(0), pentru orice numere reale x și y.

5p

c) Determinați numerele reale x pentru care A(x) · A(0) = I₂, unde I₂ = [[1, 0], [0, 1]].

5p

2. Se consideră polinomul P(X) = X³ - 3X² + mX + 4, unde m este număr real. Știind că X=1 este o rădăcină a polinomului P.

5p

a) Arătați că m = -2.

5p

b) Pentru m = -2, determinați celelalte două rădăcini ale polinomului P.

5p

c) Pentru m = -2, calculați x₁ + x₂ + x₃ + x₁x₂ + x₁x₃ + x₂x₃, unde x₁, x₂, x₃ sunt rădăcinile polinomului P.
